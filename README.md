# FractalShares
#### A simple way for tokenization of the Real World Assets


### Project SetUp

```shell
$ git clone https://github.com/jitendragangwar123/FractalShares.git
$ cd FractalShares
```
#### To start the Back-end
```shell
$ cd server
$ npm i
$ npm start
```
#### To start the Front-end
```shell
$ cd client
$ npm i
$ npm run dev
```
